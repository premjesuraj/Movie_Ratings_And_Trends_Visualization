# Movie_Ratings_And_Trends_Visualization
The project focuses on transforming raw movie-related data into meaningful visual insights using modern Python visualization libraries. The analysis includes:

- Exploratory Data Analysis (EDA)
- Statistical analysis of movie trends
- Genre-wise rating comparisons
- Revenue and rating correlation analysis
- Interactive dashboards using Plotly
- Static and interactive visualizations

The dataset contains approximately 1,000 movies released between 1970 and 2023 with attributes such as:

- Movie Title
- Genre
- Release Year
- IMDb Rating
- Votes
- Revenue
- Runtime
- Decade
